# Pantry — Chapter 2: Limits

## Scenes
- Einstein 1905 mass-velocity equation approaching c — §2.1. Reusable for any chapter on asymptotes, infinities, or relativity contexts.
- The function $(x^2-4)/(x-2)$ near $x=2$ — canonical example of limit existing where function value doesn't. §2.2.

## Etymologies
- **Limit** — from Latin *limes* (boundary). The boundary value the function approaches but may never reach.
- **Continuous** — from Latin *continuus* (uninterrupted). Functions whose graphs don't break.
- **Asymptote** — from Greek *asymptōtos* (not falling together). A line the curve approaches without touching.

## Trade-offs named
- Intuitive definition buys accessibility at the cost of precision; ε-δ buys rigor at the cost of overhead. §2.2, §2.6.
- Limit laws buy systematic computability at the cost of failing on indeterminate forms. §2.4.
- IVT buys existence proofs at the cost of not telling us where the solution is. §2.5.

## Worked examples
- $(x^2-4)/(x-2)$ → 4 by factoring. §2.2.
- $(\sqrt{x+1} - 1)/x$ → 1/2 by conjugate. §2.4.
- Squeeze: $x^2 \sin(1/x)$ → 0. §2.4.
- IVT: $x^3 - x - 1$ has root in $[1,2]$. §2.5.
- ε-δ proof: $\lim_{x\to 3}(2x+1) = 7$ with $\delta = \epsilon/2$. §2.6.

Limit; one-sided limit; two-sided limit; infinite limit; limit at infinity; indeterminate form (0/0, ∞/∞); limit laws; squeeze theorem; continuity; removable / jump / infinite discontinuity; Intermediate Value Theorem; ε-δ definition.
